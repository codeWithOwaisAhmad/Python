## 🤔 Who is This Repository For?

This repository was meticulously crafted from scratch to serve multiple audiences:

1. **Absolute Beginners:** If you've never written a line of code, follow the curriculum from `01_Variables`. Every module is explained in plain English, with extensive comments.
2. **Intermediate Developers:** Quickly look up syntax, modern "Pythonic" techniques (like list comprehensions and match-case), and advanced type hinting.
3. **Interview Candidates:** **Every single module** includes an exclusive `Interview Prep & Gotchas` section. We cover common traps, time-complexities, and deep-dive questions that Big-Tech companies love to ask.

---

## ✨ Key Features

- **Sequential Learning Path:** Folders are numbered `01` to `23` to guide you step-by-step.
- **Modern Python Standards:** All code uses Python 3.10+ features, Type Hinting, and follows PEP 8 standards.
- **Execution Ready:** Every script has an `if __name__ == "__main__":` block so you can run them instantly.
- **Gotchas and Edge Cases:** Deep explanations of *why* things work, not just *how*. (e.g., Shallow vs Deep Copies, Memory Efficiency, The Mutable Default Argument Trap).

## 📚 The Curriculum

Explore the modules below. We highly recommend proceeding in order if you are relatively new to programming.

### Part 1: The Fundamentals
*The building blocks of Python.*
- 📁 **[01_Variables](./01_Variables)** - Assignment, Scope, Global/Local variables.
- 📁 **[02_Datatypes](./02_Datatypes)** - Primitives, Object checking.
- 📁 **[03_Casting](./03_Casting)** - Implicit and Explicit casting.
- 📁 **[04_Strings](./04_Strings)** - String slicing, F-Strings, Immutability.
- 📁 **[05_String_Methods](./05_String_Methods)** - Modifying and sanitizing text.
- 📁 **[06_Operators](./06_Operators)** - Arithmetic, Logical, Bitwise.
- 📁 **[07_Precedence](./07_Precedence)** - BODMAS/PEMDAS and logical short-circuiting.

### Part 2: Control Flow
*Decision making and iterating through your code.*
- 📁 **[08_Conditionals](./08_Conditionals)** - If/Else, Ternary Operators, Match-Case.
- 📁 **[09_While_Loop](./09_While_Loop)** - Break, Continue, The While-Else clause.
- 📁 **[10_For_Loop](./10_For_Loop)** - Iterating, `range()`, `enumerate()`, List Comprehensions.

### Part 3: Data Structures (Collections)
*Storing, organizing, and accessing data efficiently.*
- 📁 **[11_Lists](./11_Lists)** - Mutable Arrays, Deep/Shallow Copy gotchas.
- 📁 **[12_Tuples](./12_Tuples)** - Immutable Arrays, Packing & Unpacking.
- 📁 **[13_Sets](./13_Sets)** - Unique elements, Set Theory (Union, Intersection), O(1) Lookups.
- 📁 **[14_Dictionaries](./14_Dictionaries)** - Hash Maps, Key-Value pairs, Dictionary Comprehensions.

### Part 4: Reusability and Logic
*Writing clean, modular code.*
- 📁 **[15_Functions](./15_Functions)** - `*args`, `**kwargs`, Default parameters.
- 📁 **[16_Recursion](./16_Recursion)** - Base cases, call stacks, Factorials vs Fibonacci.
- 📁 **[17_Exception_Handling](./17_Exception_Handling)** - Graceful degradation via `try/except/finally`.
- 📁 **[18_File_Handling](./18_File_Handling)** - Context Managers (`with open`), JSON parsing.
- 📁 **[19_Regular_Expressions](./19_Regular_Expressions)** - Pattern matching, extraction, substitution.

### Part 5: Advanced Python (Pro Level)
*The concepts that separate beginners from Senior Engineers.*
- 📁 **[20_OOP](./20_OOP)** - Object-Oriented Principles: Inheritance, Polymorphism, Magic Methods.
- 📁 **[21_Decorators](./21_Decorators)** - Wrapping functions dynamically.
- 📁 **[22_Generators_Iterators](./22_Generators_Iterators)** - Memory efficiency with the `yield` statement.
- 📁 **[23_Lambda_Functions](./23_Lambda_Functions)** - Anonymous inline functions (`map`, `filter`, `sorted`).

## 🚀 How to Use This Repository

### 1. Clone the repo
```bash
git clone https://github.com/codeWithOwaisAhmad/Python.git
cd Python
```

### 2. Navigate and Run
Every folder contains a standalone python script. To learn about Lists, for example:
```bash
cd 11_Lists
python lists.py
```
> **Tip:** Read the source code! The scripts are heavily commented and designed to be read top-to-bottom like a textbook before you run them.
